# Project Brief: Parkshare

## Core Goals
Intelligentes Parkplatz-Sharing für das Orendt Studios Team. Ermöglicht es festen Platzinhabern, freie Tage bekannt zu geben, und flexiblen Mitarbeitern, diese Plätze zu buchen.

## Scope
- User Roles: Admin, Owner (Platzinhaber), Flexible (Bucher).
- Core Features: Availability management, Booking system, Admin dashboard.
- Branding: Orendt Studios Branding (Black/White + Neon Accent).

## UI/UX Direction
The UI should be premium, clean, editorial, and consistent with the established Orendt Studios design system as seen in `orendt-survey`.
